# QR-Code_iOS_Objective-C
QR Code Scanner기능을 사용가능한 iOS앱


1. 프로젝트의 목적 및 용도
  - QR코드를 Scan하여 데이터 정보를 인식하여 프로세스 이용
  - 하이브리드 앱 기반으로 개발되어 자바스크립트에서 네이티브 호출 기능
  - Ex) 출석체크, 입장인원 관리 등


4. 프로젝트 사용 언어 및 툴
  - 사용 언어
    Objective-C
  - 사용 툴 및 버전
    Xcode  : Version 13.3 (13E113)
    
    Mac OS : macOS Monterey 12.3
